e:T7c98,<h2 id="개요"><a class="heading-link" aria-hidden="true" href="#개요">개요</a></h2>
<p>이 프로젝트는 C++ 표준 템플릿 라이브러리(STL)의 컨테이너들을 <code>ft</code> 네임스페이스에서 재구현한 것이다. 모든 컨테이너는 표준 라이브러리와 동일한 인터페이스와 동작을 제공하며, 메모리 관리, 반복자, 템플릿 프로그래밍 등의 개념을 학습할 수 있다.</p>
<h2 id="구현된-컨테이너"><a class="heading-link" aria-hidden="true" href="#구현된-컨테이너">구현된 컨테이너</a></h2>
<h3 id="시퀀스-컨테이너-sequence-containers"><a class="heading-link" aria-hidden="true" href="#시퀀스-컨테이너-sequence-containers">시퀀스 컨테이너 (Sequence Containers)</a></h3>
<h4 id="1-vector-vectorhpp"><a class="heading-link" aria-hidden="true" href="#1-vector-vectorhpp">1. <strong>vector</strong> (<code>Vector.hpp</code>)</a></h4>
<ul>
<li>동적 배열 컨테이너</li>
<li>연속된 메모리 공간에 요소 저장</li>
<li>랜덤 액세스 반복자 지원</li>
<li>성장 인자: 1.5배 (동적 확장)</li>
</ul>
<h4 id="2-list-listhpp"><a class="heading-link" aria-hidden="true" href="#2-list-listhpp">2. <strong>list</strong> (<code>List.hpp</code>)</a></h4>
<ul>
<li>이중 연결 리스트 컨테이너</li>
<li>양방향 반복자 지원</li>
<li>양쪽 끝에서 O(1) 삽입/삭제</li>
<li>특수 연산: <code>splice</code>, <code>merge</code>, <code>sort</code>, <code>reverse</code>, <code>unique</code>, <code>remove</code></li>
</ul>
<h4 id="3-deque-dequehpp"><a class="heading-link" aria-hidden="true" href="#3-deque-dequehpp">3. <strong>deque</strong> (<code>Deque.hpp</code>)</a></h4>
<ul>
<li>양방향 큐 컨테이너</li>
<li>양쪽 끝에서 O(1) 삽입/삭제</li>
<li>랜덤 액세스 반복자 지원</li>
</ul>
<h3 id="컨테이너-어댑터-container-adapters"><a class="heading-link" aria-hidden="true" href="#컨테이너-어댑터-container-adapters">컨테이너 어댑터 (Container Adapters)</a></h3>
<h4 id="4-stack-stackhpp"><a class="heading-link" aria-hidden="true" href="#4-stack-stackhpp">4. <strong>stack</strong> (<code>Stack.hpp</code>)</a></h4>
<ul>
<li>LIFO (Last In First Out) 스택</li>
<li><code>list</code> 컨테이너를 기반으로 구현</li>
<li><code>push</code>, <code>pop</code>, <code>top</code> 연산 제공</li>
</ul>
<h4 id="5-queue-queuehpp"><a class="heading-link" aria-hidden="true" href="#5-queue-queuehpp">5. <strong>queue</strong> (<code>Queue.hpp</code>)</a></h4>
<ul>
<li>FIFO (First In First Out) 큐</li>
<li><code>list</code> 컨테이너를 기반으로 구현</li>
<li><code>push</code>, <code>pop</code>, <code>front</code>, <code>back</code> 연산 제공</li>
</ul>
<h3 id="연관-컨테이너-associative-containers"><a class="heading-link" aria-hidden="true" href="#연관-컨테이너-associative-containers">연관 컨테이너 (Associative Containers)</a></h3>
<h4 id="6-map-maphpp"><a class="heading-link" aria-hidden="true" href="#6-map-maphpp">6. <strong>map</strong> (<code>Map.hpp</code>)</a></h4>
<ul>
<li>키-값 쌍을 저장하는 정렬된 연관 컨테이너</li>
<li>이진 탐색 트리(B-tree) 기반 구현</li>
<li>중복 키 허용 안 함</li>
<li>O(log n) 검색, 삽입, 삭제</li>
</ul>
<h4 id="7-set-sethpp"><a class="heading-link" aria-hidden="true" href="#7-set-sethpp">7. <strong>set</strong> (<code>Set.hpp</code>)</a></h4>
<ul>
<li>고유한 요소를 저장하는 정렬된 연관 컨테이너</li>
<li>이진 탐색 트리(B-tree) 기반 구현</li>
<li>중복 값 허용 안 함</li>
<li>O(log n) 검색, 삽입, 삭제</li>
</ul>
<h4 id="8-multimap-multimaphpp"><a class="heading-link" aria-hidden="true" href="#8-multimap-multimaphpp">8. <strong>multimap</strong> (<code>MultiMap.hpp</code>)</a></h4>
<ul>
<li><code>map</code>과 유사하지만 중복 키 허용</li>
<li>동일한 키를 가진 여러 요소 저장 가능</li>
</ul>
<h4 id="9-multiset-multisethpp"><a class="heading-link" aria-hidden="true" href="#9-multiset-multisethpp">9. <strong>multiset</strong> (<code>MultiSet.hpp</code>)</a></h4>
<ul>
<li><code>set</code>과 유사하지만 중복 값 허용</li>
<li>동일한 값을 가진 여러 요소 저장 가능</li>
</ul>
<h2 id="주요-기능"><a class="heading-link" aria-hidden="true" href="#주요-기능">주요 기능</a></h2>
<h3 id="반복자-iterators"><a class="heading-link" aria-hidden="true" href="#반복자-iterators">반복자 (Iterators)</a></h3>
<ul>
<li><strong>Forward Iterator</strong>: <code>list</code>, <code>set</code>, <code>map</code> 등</li>
<li><strong>Bidirectional Iterator</strong>: <code>list</code>, <code>set</code>, <code>map</code> 등</li>
<li><strong>Random Access Iterator</strong>: <code>vector</code>, <code>deque</code></li>
<li><strong>Reverse Iterator</strong>: 모든 컨테이너 지원</li>
<li><strong>Const Iterator</strong>: 모든 컨테이너 지원</li>
</ul>
<h3 id="메모리-관리"><a class="heading-link" aria-hidden="true" href="#메모리-관리">메모리 관리</a></h3>
<ul>
<li>커스텀 allocator 지원</li>
<li><strong>RAII 원칙 준수</strong>: 리소스 획득은 초기화(Resource Acquisition Is Initialization)
<ul>
<li>객체가 생성될 때 리소스(메모리)를 획득하고, 소멸될 때 자동으로 해제</li>
<li>소멸자에서 <code>destroy()</code>와 <code>deallocate()</code>를 호출하여 메모리 누수 방지</li>
<li>예시: <code>vector</code>의 소멸자에서 할당된 모든 요소를 <code>destroy()</code>하고 메모리를 <code>deallocate()</code></li>
</ul>
</li>
<li><strong>예외 안전성 보장</strong>: 예외 발생 시에도 리소스가 안전하게 관리된다
<ul>
<li><code>at()</code>: 범위를 벗어나면 <code>std::out_of_range</code> 예외 발생</li>
<li><code>reserve()</code>: 너무 큰 크기 요청 시 <code>std::length_error</code> 예외 발생</li>
<li>예외 발생 시에도 기존 데이터의 무결성 유지 (Basic Guarantee)</li>
</ul>
</li>
</ul>
<h3 id="템플릿-프로그래밍"><a class="heading-link" aria-hidden="true" href="#템플릿-프로그래밍">템플릿 프로그래밍</a></h3>
<ul>
<li>SFINAE (Substitution Failure Is Not An Error) 활용</li>
<li><code>enable_if</code>를 통한 템플릿 특수화</li>
<li><code>is_integral</code> 타입 트레이트</li>
</ul>

<h2 id="구현-세부사항"><a class="heading-link" aria-hidden="true" href="#구현-세부사항">구현 세부사항</a></h2>
<h3 id="vector"><a class="heading-link" aria-hidden="true" href="#vector">Vector</a></h3>
<ul>
<li><strong>메모리 할당</strong>: 동적 배열 할당</li>
<li><strong>성장 전략</strong>: 현재 크기의 1.5배로 확장</li>
<li><strong>반복자 무효화</strong>: 재할당 시 모든 반복자 무효화</li>
<li><strong>예외 안전성</strong>: 기본 보장 (Basic Guarantee)</li>
</ul>
<h3 id="list"><a class="heading-link" aria-hidden="true" href="#list">List</a></h3>
<ul>
<li><strong>구조</strong>: 이중 연결 리스트</li>
<li><strong>더미 노드</strong>: 순환 리스트 구조 (<code>_tip</code> 노드)</li>
<li><strong>특수 연산</strong>:
<ul>
<li><code>splice</code>: O(1) 리스트 병합</li>
<li><code>merge</code>: 정렬된 리스트 병합</li>
<li><code>sort</code>: 버블 정렬 알고리즘</li>
<li><code>reverse</code>: O(n) 역순 변환</li>
</ul>
</li>
</ul>
<h3 id="mapset"><a class="heading-link" aria-hidden="true" href="#mapset">Map/Set</a></h3>
<ul>
<li><strong>구조</strong>: 이진 탐색 트리 (B-tree)</li>
<li><strong>균형</strong>: AVL 트리 또는 Red-Black 트리 (구현에 따라 다름)</li>
<li><strong>정렬</strong>: 키/값의 비교 함수를 통한 자동 정렬</li>
<li><strong>반복자</strong>: 중위 순회 (in-order traversal)</li>
</ul>
<h3 id="iterator"><a class="heading-link" aria-hidden="true" href="#iterator">Iterator</a></h3>
<ul>
<li><strong>베이스 클래스</strong>:
<ul>
<li><code>BaseIter&#x3C;T></code>: vector, deque용</li>
<li><code>ListBaseIter&#x3C;T></code>: list용</li>
<li><code>BtreeBaseIter&#x3C;E, Compare, isMulti></code>: map/set용</li>
</ul>
</li>
<li><strong>역방향 반복자</strong>: <code>ReverseIterator</code>, <code>ConstReverseIterator</code> 래퍼</li>
</ul>
<h3 id="utility"><a class="heading-link" aria-hidden="true" href="#utility">Utility</a></h3>
<ul>
<li><strong>pair</strong>: 키-값 쌍 구조체</li>
<li><strong>less</strong>: 비교 함수 객체</li>
<li><strong>enable_if</strong>: SFINAE를 위한 타입 트레이트</li>
<li><strong>is_integral</strong>: 정수 타입 판별</li>
<li><strong>swap</strong>: 값 교환 함수</li>
<li><strong>distance</strong>: 반복자 간 거리 계산</li>
</ul>
