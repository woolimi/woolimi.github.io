19:T3007,<h2 id="개요"><a class="heading-link" aria-hidden="true" href="#개요">개요</a></h2>
<p>이 프로젝트는 NASM 어셈블러를 사용하여 macOS 환경에서 동작하는 기본적인 문자열 및 파일 I/O 함수들을 어셈블리 언어로 구현한 라이브러리다.</p>
<h2 id="구현된-함수들"><a class="heading-link" aria-hidden="true" href="#구현된-함수들">구현된 함수들</a></h2>
<h3 id="1-ft_strlen"><a class="heading-link" aria-hidden="true" href="#1-ft_strlen">1. <code>ft_strlen</code></a></h3>
<p>문자열의 길이를 계산하는 함수다.</p>
<ul>
<li><strong>인자</strong>: <code>const char *s</code> - 길이를 계산할 문자열</li>
<li><strong>반환값</strong>: 문자열의 길이 (size_t)</li>
</ul>
<h3 id="2-ft_strcpy"><a class="heading-link" aria-hidden="true" href="#2-ft_strcpy">2. <code>ft_strcpy</code></a></h3>
<p>문자열을 복사하는 함수다.</p>
<ul>
<li><strong>인자</strong>:
<ul>
<li><code>char *dest</code> - 복사될 목적지 버퍼</li>
<li><code>const char *src</code> - 복사할 소스 문자열</li>
</ul>
</li>
<li><strong>반환값</strong>: <code>dest</code> 포인터</li>
</ul>
<h3 id="3-ft_strcmp"><a class="heading-link" aria-hidden="true" href="#3-ft_strcmp">3. <code>ft_strcmp</code></a></h3>
<p>두 문자열을 비교하는 함수다.</p>
<ul>
<li><strong>인자</strong>:
<ul>
<li><code>const char *s1</code> - 첫 번째 문자열</li>
<li><code>const char *s2</code> - 두 번째 문자열</li>
</ul>
</li>
<li><strong>반환값</strong>:
<ul>
<li><code>s1 &#x3C; s2</code>: 음수</li>
<li><code>s1 == s2</code>: 0</li>
<li><code>s1 > s2</code>: 양수</li>
</ul>
</li>
</ul>
<h3 id="4-ft_write"><a class="heading-link" aria-hidden="true" href="#4-ft_write">4. <code>ft_write</code></a></h3>
<p>파일 디스크립터에 데이터를 쓰는 함수다.</p>
<ul>
<li><strong>인자</strong>:
<ul>
<li><code>int fd</code> - 파일 디스크립터</li>
<li><code>const void *buf</code> - 쓸 데이터 버퍼</li>
<li><code>size_t nbyte</code> - 쓸 바이트 수</li>
</ul>
</li>
<li><strong>반환값</strong>: 실제로 쓴 바이트 수, 오류 시 -1</li>
</ul>
<h3 id="5-ft_read"><a class="heading-link" aria-hidden="true" href="#5-ft_read">5. <code>ft_read</code></a></h3>
<p>파일 디스크립터에서 데이터를 읽는 함수다.</p>
<ul>
<li><strong>인자</strong>:
<ul>
<li><code>int fd</code> - 파일 디스크립터</li>
<li><code>void *buf</code> - 읽은 데이터를 저장할 버퍼</li>
<li><code>size_t nbyte</code> - 읽을 바이트 수</li>
</ul>
</li>
<li><strong>반환값</strong>: 실제로 읽은 바이트 수, 오류 시 -1</li>
</ul>
<h3 id="6-ft_strdup"><a class="heading-link" aria-hidden="true" href="#6-ft_strdup">6. <code>ft_strdup</code></a></h3>
<p>문자열을 복제하여 새로운 메모리에 저장하는 함수다.</p>
<ul>
<li><strong>인자</strong>: <code>const char *s</code> - 복제할 문자열</li>
<li><strong>반환값</strong>: 새로 할당된 메모리에 복제된 문자열의 포인터, 실패 시 NULL</li>
</ul>

<h2 id="x86-64-레지스터"><a class="heading-link" aria-hidden="true" href="#x86-64-레지스터">x86-64 레지스터</a></h2>
<h3 id="범용-레지스터-general-purpose-registers"><a class="heading-link" aria-hidden="true" href="#범용-레지스터-general-purpose-registers">범용 레지스터 (General Purpose Registers)</a></h3>
<ul>
<li><strong>RAX</strong>: 주 누산기 (primary accumulator)</li>
<li><strong>RBX</strong>: 베이스 레지스터 (base register)</li>
<li><strong>RCX</strong>: 카운트 레지스터 (count register)</li>
<li><strong>RDX</strong>: 데이터 레지스터 (data register)</li>
<li><strong>RSI</strong>: 소스 인덱스 (source index)</li>
<li><strong>RDI</strong>: 목적지 인덱스 (destination index)</li>
<li><strong>RSP</strong>: 스택 포인터 (stack pointer)</li>
<li><strong>RBP</strong>: 베이스 포인터 (base pointer)</li>
<li><strong>R8 ~ R15</strong>: 추가 범용 레지스터</li>
</ul>
<h3 id="세그먼트-레지스터-segment-registers"><a class="heading-link" aria-hidden="true" href="#세그먼트-레지스터-segment-registers">세그먼트 레지스터 (Segment Registers)</a></h3>
<ul>
<li><strong>CS</strong>: 코드 세그먼트 (code segment)</li>
<li><strong>SS</strong>: 스택 세그먼트 (stack segment)</li>
<li><strong>DS</strong>: 데이터 세그먼트 (data segment)</li>
<li><strong>ES</strong>: 확장 세그먼트 (extra segment)</li>
<li><strong>FS</strong>: 추가 세그먼트</li>
<li><strong>GS</strong>: 추가 세그먼트</li>
</ul>
