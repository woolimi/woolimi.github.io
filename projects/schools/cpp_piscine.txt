19:T3db1,<h2 id="개요"><a class="heading-link" aria-hidden="true" href="#개요">개요</a></h2>
<p>이 프로젝트는 ecole42의 C++ 교육 과정을 통해 C++ 프로그래밍의 기초부터 고급 개념까지 단계적으로 학습할 수 있도록 설계되었다. 각 모듈은 특정 C++ 개념에 집중하여 실습을 통해 이해를 돕는다.</p>
<h2 id="프로젝트-구조"><a class="heading-link" aria-hidden="true" href="#프로젝트-구조">프로젝트 구조</a></h2>

</code></pre>

<h3 id="module-00---c-기초"><a class="heading-link" aria-hidden="true" href="#module-00---c-기초">Module 00 - C++ 기초</a></h3>

<ul>

<li><strong>ex01 - phonebook</strong>: 간단한 전화번호부 프로그램 (최대 8개 연락처)</li>
<li><strong>ex02 - accounts</strong>: 계좌 클래스 구현</li>
</ul>
<p><strong>주요 학습 내용:</strong></p>
<ul>
<li>기본 입출력 (<code>std::cout</code>, <code>std::cin</code>)</li>
<li>문자열 처리</li>
<li>클래스와 객체의 기본 개념</li>
</ul>

<li><strong>ex04</strong>: 포인터와 참조 심화</li>
<li><strong>ex05 - Brain</strong>: 포인터를 사용한 객체 관계</li>
