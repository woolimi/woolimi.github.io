<h2 id="기술-스택"><a class="heading-link" aria-hidden="true" href="#기술-스택">기술 스택</a></h2>
<h3 id="backend"><a class="heading-link" aria-hidden="true" href="#backend">Backend</a></h3>
<ul>
<li><strong>Framework</strong>: NestJS</li>
<li><strong>Database</strong>: MySQL</li>
<li><strong>ORM</strong>: TypeORM</li>
<li><strong>Authentication</strong>: Passport.js (JWT, OAuth2)</li>
<li><strong>Streaming</strong>: torrent-stream, fluent-ffmpeg</li>
<li><strong>API Documentation</strong>: Swagger</li>
<li><strong>Task Scheduling</strong>: @nestjs/schedule (Cron)</li>
</ul>
<h3 id="frontend"><a class="heading-link" aria-hidden="true" href="#frontend">Frontend</a></h3>
<ul>
<li><strong>Framework</strong>: Nuxt 3</li>
<li><strong>UI Library</strong>: PrimeVue</li>
<li><strong>Styling</strong>: TailwindCSS, SCSS</li>
<li><strong>State Management</strong>: Pinia</li>
<li><strong>Internationalization</strong>: @nuxtjs/i18n (FR/EN)</li>
<li><strong>HTTP Client</strong>: Axios</li>
</ul>
<h3 id="infrastructure"><a class="heading-link" aria-hidden="true" href="#infrastructure">Infrastructure</a></h3>
<ul>
<li><strong>Containerization</strong>: Docker &#x26; Docker Compose</li>
<li><strong>Package Manager</strong>: pnpm</li>
</ul>
<h2 id="주요-기능"><a class="heading-link" aria-hidden="true" href="#주요-기능">주요 기능</a></h2>
<h3 id="인증-및-사용자-관리"><a class="heading-link" aria-hidden="true" href="#인증-및-사용자-관리">인증 및 사용자 관리</a></h3>
<ul>
<li><strong>OAuth2 로그인</strong>: 42, Google, GitHub 지원</li>
<li><strong>로컬 인증</strong>: 이메일/비밀번호 기반 회원가입 및 로그인</li>
<li><strong>비밀번호 재설정</strong>: 이메일을 통한 비밀번호 재설정 기능</li>
<li><strong>프로필 관리</strong>: 사용자 정보 수정 및 프로필 이미지 업로드</li>
</ul>

<ul>

<li><strong>다양한 해상도 지원</strong>: 여러 비디오 품질 옵션 제공</li>
