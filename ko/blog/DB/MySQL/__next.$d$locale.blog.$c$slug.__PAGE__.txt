<li><strong>FROM</strong> — 테이블 선택</li>
<li><strong>WHERE</strong> — 행 필터</li>
<li><strong>GROUP BY</strong> — 그룹화</li>
<li><strong>HAVING</strong> — 그룹 필터</li>
<li><strong>SELECT</strong> — 컬럼/집계 계산</li>
<li><strong>ORDER BY</strong> — 정렬</li>

<ul>

<li>두 번째 SELECT: RIGHT JOIN으로 오른쪽 테이블 전체 + 왼쪽 매칭 행</li>
<li>UNION: 두 결과를 합치면서 중복 제거 (양쪽 모두 존재하는 행은 한 번만 포함)</li>
</ul>
<p><strong>참고</strong>: 중복을 허용하려면 <code>UNION ALL</code>을 사용할 수 있지만, FULL OUTER JOIN의 의미상 <code>UNION</code>을 사용하는 것이 일반적이다.</p>

<h3 id="dcl-data-control-language---데이터-제어-언어"><a class="heading-link" aria-hidden="true" href="#dcl-data-control-language---데이터-제어-언어">DCL (Data Control Language) - 데이터 제어 언어</a></h3>
