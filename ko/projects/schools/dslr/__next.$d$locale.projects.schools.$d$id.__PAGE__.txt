<li><strong>Batch Gradient Descent</strong>: 전체 데이터셋을 사용하여 한 번에 업데이트</li>
<li><strong>Mini-Batch Gradient Descent</strong>: 작은 배치 단위로 업데이트 (기본 배치 크기: 5)</li>
<li><strong>Stochastic Gradient Descent</strong>: 각 샘플마다 업데이트</li>
</ul>

<h2 id="주요-기능"><a class="heading-link" aria-hidden="true" href="#주요-기능">주요 기능</a></h2>
<h3 id="1-데이터-전처리-01describepy"><a class="heading-link" aria-hidden="true" href="#1-데이터-전처리-01describepy">1. 데이터 전처리 (01.describe.py)</a></h3>
<ul>
<li>CSV 파일을 읽고 데이터셋의 통계 정보를 출력한다</li>
<li>각 컬럼의 타입, 개수, 고유값, 최빈값, 빈도, 평균, 최소값, 사분위수, 최대값, 표준편차를 계산한다</li>
</ul>
<h3 id="2-히스토그램-02histogrampy"><a class="heading-link" aria-hidden="true" href="#2-히스토그램-02histogrampy">2. 히스토그램 (02.histogram.py)</a></h3>
<ul>
<li>각 특성별로 기숙사 간 분포를 히스토그램으로 시각화한다</li>
<li><code>--all</code> 옵션으로 모든 히스토그램을 한 번에 생성할 수 있다</li>
</ul>
<h3 id="3-산점도-03scatter_plotpy"><a class="heading-link" aria-hidden="true" href="#3-산점도-03scatter_plotpy">3. 산점도 (03.scatter_plot.py)</a></h3>
<ul>
<li>특성 간 상관관계를 산점도로 시각화하고 피어슨 상관계수를 제공한다</li>
<li><code>--all</code> 옵션으로 모든 산점도를 생성할 수 있다</li>
</ul>
<h3 id="4-페어-플롯-04pair_plotpy"><a class="heading-link" aria-hidden="true" href="#4-페어-플롯-04pair_plotpy">4. 페어 플롯 (04.pair_plot.py)</a></h3>
<ul>
<li>모든 특성 쌍의 관계를 한 번에 시각화한다</li>
<li>특성 선택 및 제거 결정에 도움을 준다</li>
</ul>
<h3 id="5-모델-학습-05logreg_trainpy"><a class="heading-link" aria-hidden="true" href="#5-모델-학습-05logreg_trainpy">5. 모델 학습 (05.logreg_train.py)</a></h3>
<ul>
<li>선택된 특성을 사용하여 로지스틱 회귀 모델을 학습한다</li>
<li>각 기숙사(Gryffindor, Hufflepuff, Ravenclaw, Slytherin)에 대해 이진 분류기를 학습한다</li>
<li>학습된 가중치를 <code>weights.csv</code>에 저장한다</li>
<li>학습 손실 그래프를 생성한다</li>
</ul>
<h3 id="6-예측-06logreg_predictpy"><a class="heading-link" aria-hidden="true" href="#6-예측-06logreg_predictpy">6. 예측 (06.logreg_predict.py)</a></h3>
<ul>
<li>학습된 모델을 사용하여 테스트 데이터셋의 기숙사를 예측한다</li>
<li>예측 결과를 <code>houses.csv</code>에 저장한다</li>
</ul>
<h3 id="7-평가-07evaluatepy"><a class="heading-link" aria-hidden="true" href="#7-평가-07evaluatepy">7. 평가 (07.evaluate.py)</a></h3>
<ul>
<li>예측 결과와 정답을 비교하여 모델의 정확도를 계산한다</li>
</ul>
