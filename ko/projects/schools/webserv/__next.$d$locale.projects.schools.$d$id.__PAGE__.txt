e:T8f35,<h2 id="개요"><a class="heading-link" aria-hidden="true" href="#개요">개요</a></h2>
<p>WebServ는 ecole42의 프로젝트로, C++을 사용하여 HTTP/1.1 표준을 준수하는 웹서버를 처음부터 구현한 프로젝트다. 이 프로젝트를 통해 소켓 프로그래밍, 멀티플렉싱, HTTP 프로토콜, CGI 등의 개념을 학습한다.</p>
<h2 id="주요-기능"><a class="heading-link" aria-hidden="true" href="#주요-기능">주요 기능</a></h2>
<h3 id="1-멀티플렉싱"><a class="heading-link" aria-hidden="true" href="#1-멀티플렉싱">1. 멀티플렉싱</a></h3>
<ul>
<li><code>select()</code> 시스템 콜을 사용한 I/O 멀티플렉싱</li>
<li>단일 스레드에서 여러 클라이언트 동시 처리</li>
<li>읽기/쓰기 이벤트 동시 모니터링</li>
</ul>
<h3 id="2-설정-파일-파싱"><a class="heading-link" aria-hidden="true" href="#2-설정-파일-파싱">2. 설정 파일 파싱</a></h3>
<ul>
<li>Nginx 스타일의 설정 파일 지원</li>
<li>여러 서버 블록 설정 가능</li>
<li>각 서버별 포트, 호스트명, 루트 디렉토리 설정</li>
<li>Location 블록을 통한 라우팅 설정</li>
</ul>
<h3 id="3-http-메서드-지원"><a class="heading-link" aria-hidden="true" href="#3-http-메서드-지원">3. HTTP 메서드 지원</a></h3>
<ul>
<li>GET: 리소스 조회</li>
<li>POST: 리소스 생성 및 데이터 전송</li>
<li>PUT: 리소스 업데이트</li>
<li>DELETE: 리소스 삭제</li>
<li>OPTIONS: 지원하는 메서드 확인</li>
<li>TRACE: 요청 추적</li>
</ul>
<h3 id="4-cgi-지원"><a class="heading-link" aria-hidden="true" href="#4-cgi-지원">4. CGI 지원</a></h3>
<ul>
<li>PHP, Python 등 스크립트 언어 실행</li>
<li>확장자별 CGI 프로그램 매핑</li>
<li>환경 변수 전달 및 표준 입출력 처리</li>
</ul>
<h3 id="5-파일-업로드"><a class="heading-link" aria-hidden="true" href="#5-파일-업로드">5. 파일 업로드</a></h3>
<ul>
<li>POST/PUT 메서드를 통한 파일 업로드</li>
<li>업로드 폴더 지정 가능</li>
<li>클라이언트 본문 크기 제한</li>
</ul>
<h3 id="6-에러-처리"><a class="heading-link" aria-hidden="true" href="#6-에러-처리">6. 에러 처리</a></h3>
<ul>
<li>HTTP 상태 코드 처리</li>
<li>커스텀 에러 페이지 설정</li>
<li>타임아웃 처리 (클라이언트: 30초, 서버: 3초)</li>
</ul>
<h3 id="7-디렉토리-리스팅"><a class="heading-link" aria-hidden="true" href="#7-디렉토리-리스팅">7. 디렉토리 리스팅</a></h3>
<ul>
<li>autoindex 옵션을 통한 디렉토리 목록 표시</li>
<li>index 파일 자동 검색</li>
</ul>

<h3 id="서버-블록-속성"><a class="heading-link" aria-hidden="true" href="#서버-블록-속성">서버 블록 속성</a></h3>
<ul>
<li><code>listen</code>: 포트 번호 (필수)</li>
<li><code>server_name</code>: 서버 이름</li>

<ul>
<li><code>root</code>: Location별 루트 디렉토리</li>
<li><code>autoindex</code>: 디렉토리 리스팅 활성화 (<code>on</code>/<code>off</code>)</li>
<li><code>index</code>: 기본 인덱스 파일 목록</li>
<li><code>allow</code>: 허용할 HTTP 메서드 목록</li>
<li><code>cgi</code>: CGI 설정 (<code>확장자 실행파일경로</code>)</li>
<li><code>upload_folder</code>: 파일 업로드 폴더 경로</li>
<li><code>client_max_body_size</code>: Location별 본문 크기 제한</li>
</ul>
